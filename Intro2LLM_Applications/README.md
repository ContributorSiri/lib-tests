# Introduction to Large Language Model Applications (计算机系统概论)

- **Instructor**: Tianxing He (贺天行, 2026), Wei Xu (徐葳, 2025)
- This course is for freshmen in spring semester.
- **Course Description**: As a professional elective course for new students with almost zero AI background, this course aims to enable information-related majors to initially grasp the unique R&D models, application scenarios, and practical hands-on coding skills of large language models (LLMs). It covers: the few-shot learning capabilities of LLMs, end-to-end development skills based on large models, LLM fine-tuning abilities, and data annotation and organization skills. The course consists of lectures and extensive programming practices. Under the guidance of teaching assistants, students will complete several independent small experiments and work in teams to develop a real-world application of large language models. Through this course, students will learn to: Master methods for inference and few-shot learning using pre-trained large language models; Acquire LLM fine-tuning skills, proficiency in distributed training systems, and methods for evaluating LLM quality; Develop end-to-end practical development capabilities for LLM applications, completing the end-to-end development and demonstration of an LLM application project; Understand the latest application scenarios of large language models and cutting-edge research issues; Enhance teamwork skills and the ability to present application projects.
