## How to Contribute ğŸ¤

We welcome everyone in IIIS to upload **exam papers, answers, and cheatsheets**! Every single file is helpful! ğŸ’¡

> **Q:** What is a _cheatsheet_?  
> **A:** In some exams in IIIS, students are allowed to bring some paper reference materials related to the exam, which are called cheatsheets. Because some courses have restrictions on cheatsheets (for example, only one A4 page is allowed), it is necessary to make your own cheatsheet. So, we hope it can be shared too! ğŸ“„

To maintain a great structure of the repo, we recommend you check the following before submitting a _pull request_:

- Your exam papers or answers should be in either `.md` (recommended), `.tex`, or `.pdf` (not recommended but acceptable) format.
    - **IMPORTANT**: If you want to upload an exam answer, please clarify it in the name of the file (e.g., `2025_spring_midterm-answer.md`). If your answer is unofficial, please state it clearly in your file.
- All images should be added correctly in `./Images`. They should not appear in the subfolders of any course.
- You may also update `README.md` to reflect course descriptions, instructors, and logistics for each year. We recommend writing a `README.md` (refer to other `README` files for the format) when adding a new course.
- For cheatsheets, `.docx` (not recommended) and `.pdf` are also acceptable. We highly recommend you check the following:
    - Upload your cheatsheet inside the `cheatsheet/` subfolder in the folder of the correct course. 
    - Name your cheatsheet file as `cs_<year>_<name>.<file_type>` (e.g., `midterm_cs_2024_jason.tex` or `cs_2023_yqz.md`).
    - Update the `README.md` in the course folder to record your contribution.
- **DO NOT UPLOAD**:
    - $\LaTeX$ compiling files `.aux`, `.log`, etc.;
    - Textbooks (which may be too large);
    - Other unrelated stuff.
